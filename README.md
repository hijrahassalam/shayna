# Shayna Online Shop

Project Aplikasi Toko Online sebagai bahan belajar Full Stack Developer, dikembangkan dengan VueJS dan Laravel

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
